TP_1 - Uma máquina virtual que simula um processador simples, e usa instruções e registradores para funções básicas como multiplicação.
compilar e executar:
gcc *.c -o Tp1
./Tp1



TP_2 - A máquina um pouco mais aprimorada, com hierarquia de memória implementada, com cache e RAM. Usa um Gerador de Instruções para gerar um arquivo de texto, com diferentes métricas, para testar o desempenho
compilar e executar:
gcc *.c -o Tp2
./Tp2



TP_3 - Semelhante ao projeto anterior, mas com armazenamento em disco implementado, como um arquivo binário.
compilar e executar:
gcc *.c -o Tp3
./Tp3
